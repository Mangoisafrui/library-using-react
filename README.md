# Open Library Web App(react)
A React web application that lets users search and explore books using the Open Library API.

## Features
- Fetches book data from the Open Library API using Axios
- Displays book titles, descriptions, and cover images
- Live search filtering to find books by title or author
- React Router navigation between pages
- Responsive layout built with React Bootstrap
- Error handling for failed API requests

## Technologies
- React
- Vite
- React Router
- Axios
- React Bootstrap
- Open Library API
- JavaScript

## API
- Open Library: `https://openlibrary.org/`

## Components
- **Header** — app title and branding
- **Nav** — navigation links between pages using React Router
- **Footer** — app information and links
- **BookList** — fetches and displays the list of books
- **BookCard** — renders individual book data including cover image

## How to Run
```bash
npm install
npm run dev
```
